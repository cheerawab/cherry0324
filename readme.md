## 🔧 設定環境變數（若你是開發者想使用此機器人）

請在 `.env` 檔案中設定以下變數，以確保機器人正常運作：
```
DISCORD_TOKEN=
CLIENT_ID=
ALLOWED_CHANNEL_ID=
SUPPORT_ROLE_ID=
TICKET_LOG_CHANNEL_ID=
GEMINI_API_KEY=
SELFINTRODUCTIONCHANNEL=
```
📌 **請務必將 `.env` 檔案加入 `.gitignore`，避免公開敏感資訊。**
